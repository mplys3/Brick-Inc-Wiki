# Brick Inc Wiki

Static wiki extracted from the local Brick Inc. Idle Breaker APK data.

## View locally

Open `docs/index.html` in a browser.

## GitHub Pages

This repository is prepared for GitHub Pages from the `/docs` folder.

In GitHub:

1. Open **Settings**.
2. Go to **Pages**.
3. Select **Deploy from a branch**.
4. Choose the default branch and the `/docs` folder.

## Rebuild

The generated site can be rebuilt with:

```powershell
python .\tools\build_full_site.py
```
